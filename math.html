The following is an expository paper I wrote as part of a Combinatorics class exploring graph graph partitions. Utilizing the Laplacian matrix of a graph and analyzing the eigenvalues and eigenvectors, someone can holistically determine well/highly connected subgraphs and partition the graph accordingly. This is one of the earliest papers I have written as an undergraduate. Special thanks to my friend Justin Berke for creating a graph program that remains useful to this day.
